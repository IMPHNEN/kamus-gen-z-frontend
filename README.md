## Bagaimana caranya?

Untuk member organization IMPHNEN

1. Clone repository ini
2. Buat branch baru dengan nama user
3. Commit dan push
4. Buat pull request

Untuk non member organization IMPHNEN

1. Fork repository
2. Clone repository
3. Commit dan push
4. Buat pull request
